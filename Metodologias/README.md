## Meus Projetos

### Em 2025-1
O projeto desenvolvido pela equipe VORTEK consistiu no Sistema de Ponto, uma aplicação web focada no controle de jornada de trabalho de colaboradores de empresas terceirizadas. A plataforma oferece cadastro de empresas e profissionais, registro de ponto, geração de relatórios, dashboards com filtragem por data e extração de dados, tudo com usabilidade e segurança em mente

O projeto foi proposto pela empresa ALTAVE, que tinha a necessidade de otimizar o registro e o acompanhamento das horas trabalhadas por funcionários terceirizados. A plataforma tem como objetivo trazer precisão aos dados e aumentar a eficiência na administração de equipes

O principal problema identificado foi a falta de um sistema integrado para gerenciar o ponto dos colaboradores, o que resultava em retrabalho, inconsistência de informações e dificuldade no controle gerencial. A ausência de relatórios filtráveis por data, empresa e profissional tornava as análises lentas e pouco confiáveis, prejudicando a tomada de decisões

A solução entregue foi um sistema web completo com interface e backend articulados, que inclui:
Frontend: interface para cadastro e visualização de empresas e profissionais, upload de foto, filtros por data, exportação de relatórios, dashboards com gráficos dinâmicos e interativos 
GitHub
Backend (API): uma API desenvolvida em Java, com autenticação de usuários, armazenamento de registros de ponto, geração de relatórios, consulta histórica de horas trabalhadas, endpoints para cadastro e busca de empresas e profissionais 
GitHub

O sistema foi bem estruturado para permitir manutenção futura, escalabilidade e uma experiência fluida entre os módulos da aplicação

[GIT](https://github.com/Vortek-API)
